[cssnext](http://cssnext.io/) compatible colors for [Google's material design](https://www.google.com/design/spec/style/color.html).

### Usage

```
.regular-colors {
  color: var(--md-orange-400);
}

.accent-colors {
  color: var(--md-orange-a400);
}
```
